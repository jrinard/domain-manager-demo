# Demo assets

Static images for the portfolio demo. Served at `/demo-assets/...`.

Re-download from Cherry (one-time): `./scripts/download-demo-assets.sh`

**Folders:**
- `bg-images/` — layout builder preset backgrounds
- `logos/` — domain logo light/dark
- `menu-icons/` — navigation icons
- `sections/` — **layout section images** (featured content, banners, etc.)
- `profile/` — avatar thumb
- `trait-icons/` — custom tab icons
