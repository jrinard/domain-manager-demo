## Falcon UI
- **Fix** `SkeletonSquare` now has aria loading labeling thereby allowing us to use it just like `SkeletonText`

## Falcon Table - 03-08-24
- **ADD** `columns` now has `defaultVisibility` property for setting the default visibility of a given column (see story)

- **Add** `emptyMessage` property and story allowing developers to customize the message when the list is empty
