# md-wysiwyg

This library was generated with [Nx](https://nx.dev).

Mermaid code blocks in the editor require both **`mermaid`** and **`@spacedock/md-viewer`** at runtime (the library build marks them `external`). Both are declared as `peerDependencies`; add them to the host app.

## Running unit tests

Run `nx test md-wysiwyg` to execute the unit tests via [Vitest](https://vitest.dev/).
