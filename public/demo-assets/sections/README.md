# Section images

Drop images here for **Tryyb / Mastery layout sections** (welcome backgrounds, featured content, carousels, etc.).

Served at `/demo-assets/sections/your-file.jpg`.

Suggested naming:
- `welcome-hero.jpg`
- `featured-1.jpg`, `featured-2.jpg`
- `banner-wide.png`

After adding files, reference them in config (`src/mocks/fixtures/configBodies.ts`) or section `metadata.bgImage` / attachment fields using `/demo-assets/sections/...`.
